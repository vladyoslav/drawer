# Overlay

A layer that covers the inert portion of the view when the drawer is open.

```tsx copy
import { useState } from 'react'

import { Drawer, type Snap } from '@vladyoslav/drawer'

// or import { DrawerOverlay } from '@vladyoslav/drawer'

export const MyComponent = () => {
  const snapPoints = ['100px', '50%', '100%']
  const [snap, setSnap] = useState<Snap>(snapPoints[0])

  return (
    <Drawer.Root snap={snap} setSnap={setSnap}>
      <Drawer.Trigger />
      <Drawer.Portal>
        <Drawer.Overlay
          className="..."
          radixPrimitive={false} // use custom primitive
          fadeFrom={'100px'} // same as fadeFrom={100}
          blockInteraction={snap === '100%'} // block page scrolling on the last snap point
        />
        ...
      </Drawer.Portal>
    </Drawer.Root>
  )
}
```

## API

The `Overlay` component takes all props from [`@radix-ui/react-dialog`](https://www.radix-ui.com/primitives/docs/components/dialog#overlay) `Overlay` and some additional ones:

<Table>
  | Prop | Type | Default |
  |----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------------------|---------------------------|
  | `asChild`
  <Info>
    [`@radix-ui/react-dialog`](https://www.radix-ui.com/primitives/docs/components/dialog#overlay)
    prop <br />
    <br /> [Read the description](https://www.radix-ui.com/primitives/docs/components/dialog#overlay)
  </Info>{' '}
  | `boolean` | `false` | | `forceMount`
  <Info>
    [`@radix-ui/react-dialog`](https://www.radix-ui.com/primitives/docs/components/dialog#overlay)
    prop <br />
    <br /> [Read the description](https://www.radix-ui.com/primitives/docs/components/dialog#overlay)
  </Info>{' '}
  | `boolean` | | | `radixPrimitive`
  <Info>
    Specifies whether to use the Radix `Overlay` primitive or custom one.
    <br />
    <br />
    With `radixPrimitive={false}`, the overlay element will be in the DOM, even if
    the `modal` prop is set to `false` in `Root`, unlike the Radix one.
  </Info>{' '}
  | `boolean` | `true` | | `blockInteraction`
  <Info>
    Specifies whether to block page scrolling or not.
    <br />
    <br />
    **Available only when `radixPrimitive` is set to `false`.**
  </Info>{' '}
  | `boolean` | `Root` `modal` prop value | | `fadeFrom`
  <Info>The `y` value from which the component should fade the overlay.</Info> |
  `number \| string` | `0` | | `finalOpacity`
  <Info>
    The final opacity value for the component when it reaches the last snap
    point (or just the open state).
    <br />
    <br />
    The default value is less than 1 to make the overlay even darker when the user
    moves the drawer out of bounds.
  </Info>{' '}
  | `number` | `0.8` |
</Table>
