# Description

An accessible title to be announced when the dialog is opened.

```tsx copy
import { Drawer } from '@vladyoslav/drawer'

// or import { DrawerDescription } from '@vladyoslav/drawer'

export const MyComponent = () => (
  <Drawer.Root>
    <Drawer.Trigger />
    <Drawer.Portal>
      <Drawer.Content>
        <Drawer.Close />
        <Drawer.Title />
        <Drawer.Description>Drawer Description</Drawer.Description>
        ...
      </Drawer.Content>
    </Drawer.Portal>
  </Drawer.Root>
)
```

## API

The `Description` component has been completely re-exported from [`@radix-ui/react-dialog`](https://www.radix-ui.com/primitives/docs/components/dialog#description)

[Read Radix UI Docs](https://www.radix-ui.com/primitives/docs/components/dialog#description)
