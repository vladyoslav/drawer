# Close

The button that closes the dialog.

```tsx copy
import { Drawer } from '@vladyoslav/drawer'

// or import { DrawerClose } from '@vladyoslav/drawer'

export const MyComponent = () => (
  <Drawer.Root>
    <Drawer.Trigger />
    <Drawer.Portal>
      <Drawer.Content>
        <Drawer.Close>X</Drawer.Close>
        ...
      </Drawer.Content>
    </Drawer.Portal>
  </Drawer.Root>
)
```

## API

The `Close` component has been completely re-exported from [`@radix-ui/react-dialog`](https://www.radix-ui.com/primitives/docs/components/dialog#close)

[Read Radix UI Docs](https://www.radix-ui.com/primitives/docs/components/dialog#close)
