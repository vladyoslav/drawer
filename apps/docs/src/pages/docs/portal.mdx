# Portal

Portals your overlay and content parts into the `body`.

```tsx copy
import * as Drawer from '@vladyoslav/drawer'

// or import { DrawerPortal } from '@vladyoslav/drawer'

export const MyComponent = () => (
  <Drawer.Root>
    <Drawer.Trigger />
    <Drawer.Portal>...</Drawer.Portal>
  </Drawer.Root>
)
```

## API

The `Portal` component takes all props from [`@radix-ui/react-dialog`](https://www.radix-ui.com/primitives/docs/components/dialog#portal)

[Read Radix UI Docs](https://www.radix-ui.com/primitives/docs/components/dialog#portal)
