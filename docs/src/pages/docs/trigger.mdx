# Trigger

The button that opens the drawer.

```tsx copy
import { Drawer } from '@vladyoslav/drawer'
// or import { DrawerTrigger } from '@vladyoslav/drawer'

export const MyComponent = () => (
  <Drawer.Root>
    <Drawer.Trigger>Open Drawer</Drawer.Trigger>
    ...
  </Drawer.Root>
)
```

## API

The `Trigger` component has been completely re-exported from [`@radix-ui/react-dialog`](https://www.radix-ui.com/primitives/docs/components/dialog#trigger)

[Read Radix UI Docs](https://www.radix-ui.com/primitives/docs/components/dialog#trigger)
