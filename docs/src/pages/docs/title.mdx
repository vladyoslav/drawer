# Title

An accessible title to be announced when the dialog is opened.

```tsx copy
import { Drawer } from '@vladyoslav/drawer'
// or import { DrawerTitle } from '@vladyoslav/drawer'

export const MyComponent = () => (
  <Drawer.Root>
    <Drawer.Trigger />
    <Drawer.Portal>
      <Drawer.Content>
        <Drawer.Close />
        <Drawer.Title>Drawer Title</Drawer.Title>
        ...
      </Drawer.Content>
    </Drawer.Portal>
  </Drawer.Root>
)
```

## API

The `Title` component has been completely re-exported from [`@radix-ui/react-dialog`](https://www.radix-ui.com/primitives/docs/components/dialog#title)

[Read Radix UI Docs](https://www.radix-ui.com/primitives/docs/components/dialog#title)
