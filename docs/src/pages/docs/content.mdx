Contains content to be rendered in the open drawer.

```tsx copy
import { Drawer } from '@vladyoslav/drawer'
// or import { DrawerContent } from '@vladyoslav/drawer'

export const MyComponent = () => (
  <Drawer.Root>
    <Drawer.Trigger />
    <Drawer.Portal>
      <Drawer.Overlay />
      <Drawer.Content>
        ...
      </Drawer.Content>
    </Drawer.Portal>
  </Drawer.Root>
)
```

## API

The `Content` component takes all props from [`@radix-ui/react-dialog`](https://www.radix-ui.com/primitives/docs/components/dialog#content)

[Read Radix UI Docs](https://www.radix-ui.com/primitives/docs/components/dialog#content)
